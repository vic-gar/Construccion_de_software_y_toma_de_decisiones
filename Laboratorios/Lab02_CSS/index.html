Lab Introducción a CSS

En este laboratorio se aplican los conceptos básicos de CSS para dar estilo a una landing page,
utilizando selectores, box model, flexbox, unidades relativas y diseño responsivo.

Ver temas

Selectores CSS

Se utilizan selectores de etiqueta, clase, ID, pseudo-clase y descendiente para aplicar estilos visibles a distintos elementos de la página.

Box Model y espaciado

Las tarjetas aplican padding, margin, border y border-radius para comprender el modelo de caja y observar su comportamiento en DevTools.

Flexbox y responsividad

Se implementa Flexbox en la navegación, el hero y las tarjetas, además de media queries con enfoque mobile-first para adaptar el diseño a distintas pantallas.

Reflexión

¿Cuál es la diferencia entre padding y margin?

El padding es el espacio interno entre el contenido y el borde de un elemento y el margin es el espacio externo que separa un elemento de otros elementos.

¿Por qué se recomienda usar box-sizing: border-box?

Porque permite que el ancho y alto de un elemento incluyan el padding y el borde, lo que facilita controlar mejor el tamaño total de cada caja.

¿Qué significa diseñar con enfoque mobile-first?

Significa comenzar diseñando para dispositivos móviles y después adaptar la interfaz a pantallas más grandes usando media queries.

¿En qué situaciones preferirías rem sobre px?

Preferiría usar rem en tamaños de texto y espaciados porque permite un diseño más flexible y adaptable y usaría px en bordes o sombras donde se necesita una medida más precisa.
